Sobre o Workshop

A internet das coisas conecta milhares de dispositivos e já trouxe uma grande mudança na nossa vida. A linguagem Python está entre as 5 linguagens mais populares, segundo pesquisa da Redmonk e, por sua característica de exigir poucas linhas de código e permitir uma leitura fácil do código, ficou popular também na área de Internet das Coisas.

Nesse workshop você construirá o seu assistente pessoal para executar várias tarefas do dia a dia e também controlar seus dispositivos IoT O assistente pessoal recebe os comandos de voz, interpreta e executa as tarefas, tudo desenvolvido com Python 3.

Para quem é esse curso?

EEste curso é para quem deseja construir um projeto em python utilizando módulos de reconhecimento de voz e também o protocolo mqtt utilizado no IoT. Ao final do projeto seu assistente está pronto para executar várias tarefas personalizadas.
